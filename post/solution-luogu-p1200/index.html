【题解】P1200 [USACO1.1] 你的飞碟在这儿 Your Ride Is Here

· 2020 年 10 月 12 日 · #题解

题面

题目描述

众所周知，在每一个彗星后都有一只 UFO。这些 UFO 时常来收集地球上的忠诚支持者。不幸的是，他们的飞碟每次出行都只能带上一组支持者。因此，他们要用一种聪明的方案让这些小组提前知道谁会被彗星带走。他们为每个彗星起了一个名字，通过这些名字来决定这个小组是不是被带走的那个特定的小组（你认为是谁给这些彗星取的名字呢？）。关于如何搭配的细节会在下面告诉你；你的任务是写一个程序，通过小组名和彗星名来决定这个小组是否能被那颗彗星后面的 UFO 带走。

小组名和彗星名都以下列方式转换成一个数字：最终的数字就是名字中所有字母的积，其中是，是。例如，小组就是。如果小组的数字等于彗星的数字，你就得告诉这个小组需要准备好被带走！（记住是除以的余数；）

写出一个程序，读入彗星名和小组名并算出用上面的方案能否将两个名字搭配起来，如果能搭配，就输出“GO”，否则输出“STAY”。小组名和彗星名均是没有空格或标点的一串大写字母（不超过个字母）。

输入格式

第 1 行：一个长度为到的大写字母串，表示彗星的名字。

第 2 行：一个长度为到的大写字母串，表示队伍的名字。

输出格式

无。

思路

按照题意计算即可。

代码

#include <bits/stdc++.h>

using namespace std;

int main() {
    int us = 1, zs = 1;
    string a, b;
    getline(cin, a);
    getline(cin, b);
    for (int i = 0; i < 7; i++) {
        if (a[i] <= 0) break;
        else us *= a[i] - 64;
    }
    for (int i = 0; i < 7; i++) {
        if (b[i] <= 0) break;
        else zs *= b[i] - '@';
    }
    us %= 47;
    zs %= 47;
    if (us == zs) cout << "GO" << endl;
    else cout << "STAY" << endl;
    return 0;
}
